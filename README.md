# MN-RakSAMP

**MN-RakSAMP** is a cross-platform **SA:MP client** built with **Flutter** and a native **C++ core**.  
It is designed for the **Arsacia Game SA:MP server** to simplify login and improve user experience.  
The client supports managing multiple accounts efficiently while **minimizing resource usage**.

### Features

- ✅ Multi-account management
- ✅ Background connection service for each account
- ✅ Automatic login
- ✅ Resource-efficient (low CPU & memory usage)
